# 🚀 Astroly - Space Landing Page

Astroly is a modern astronomy-themed landing page created to explore front-end development using React, TypeScript and Tailwind CSS.
This project focuses on smooth navigation, interactive UI and a visually immersive space theme.

---

## ✨ Features

- 🌌 Animated star background
- 🪐 Planet explorer section
- 👨‍🚀 Astronaut themed UI
- 🖼 Image gallery
- 📊 Space statistics section
- 🔗 Smooth scrolling navigation
- 📱 Fully responsive design

---

## 🛠 Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite
